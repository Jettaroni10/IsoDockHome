Update instructions

To add images/take away images, access to the original powerpoint file is required.

Make your edits in powerpoint, and export the respective slides you have modified as a PNG file.

Afterwards, place the image files in the appropriate folder where the presentation is contained.

When asked if you want to replace file(s) within the origin folder, select replace (all).
